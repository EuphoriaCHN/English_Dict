## Clone 

```shell
git clone https://github.com/EuphoriaCHN/English_Dict.git
cd ./English_Dict
```

## Install Dependencies

```shell
npm config set registry https://registry.npm.taobao.org/
npm install -g yarn
yarn deps
```

## Development
